== What's this ==

This is Greasmonkey addon for Mozilla Prism(http://prism.mozilla.com/)
It's enable to use user scripts on Prism.

== Features ==

* can make user scripts to work
* can download/install user scripts from the other domain
  tools -> Greasemonkey -> Download User Script...

== How to build ==

* sh build.sh

== How to install user scripts ==

1. open dialog
   tools -> Greasemonkey -> Download User Script...
2. Input URL
